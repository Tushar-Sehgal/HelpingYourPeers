Dependency Injection in Python

In Python, DI can be implemented using simple patterns or with the help of libraries like Flask-Injector.

Example:

class Repository:
    def save(self):
        print("Save logic here")

class Service:
    def __init__(self, repository):
        self.repository = repository

    def execute(self):
        self.repository.save()

# Usage
repository = Repository()
service = Service(repository)
service.execute()
