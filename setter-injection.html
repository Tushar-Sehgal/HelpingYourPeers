Setter Injection

Setter Injection uses setter methods to inject dependencies. This method allows changing dependencies at runtime but can leave the object in an invalid state if dependencies are not set.

Example in Java:

public class Service {
    private IRepository repository;

    public void setRepository(IRepository repository) {
        this.repository = repository;
    }

    public void execute() {
        repository.save();
    }
}
